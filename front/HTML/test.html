HTML이란 무엇인가?

  1. HTML의 역할

    2. - HTML은 웹 페이지의 구조를 만드는 언어입니다.
    제목, 문단, 이미지, 목록과 같은 요소들을 정의하여 브라우저가 내용을 올바르게 표시하도록 합니다.

    - HTML 자체는 디자인을 담당하지 않으며, 웹 페이지의 뼈대 역할을 한다고 볼 수 있습니다.

  2. 웹 브라우저와 HTML

    3. - 웹 브라우저는 HTML 문서를 읽어 화면에 표시합니다.
    태그의 의미에 따라 제목은 크게, 문단은 일반 텍스트로 나타납니다.

시맨틱 태그란?

  1. 시맨틱 태그

    2. - 시맨틱 태그는 의미를 가지는 HTML 태그입니다.
    단순히 모양을 위한 태그가 아니라, 해당 영역이 어떤 역할을 하는지 명확히 나타냅니다.

    예를 들어 header는 상단 영역, nav는 메뉴 영역, footer는 하단 정보를 의미합니다.

  2. 시맨틱 태그를 사용하는 이유

    3. - 문서 구조를 이해하기 쉬워집니다.

    - 검색 엔진이 내용을 잘 파악할 수 있습니다.

    - 협업 시 코드 가독성이 좋아집니다.
